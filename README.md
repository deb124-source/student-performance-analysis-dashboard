# 🚀 Overview

The **Student Performance Analysis Dashboard** transforms raw academic data into meaningful insights using Python-based data analysis and visualization tools.

It helps educators and analysts:

- Identify high-performing students
- Detect underperforming students
- Understand attendance impact on marks
- Analyze student engagement behavior

---

## ✨ Features

### 📊 Academic Performance Analysis
- Average marks comparison
- Top vs low performers visualization
- Subject-wise performance trends

### 📈 Attendance Insights
- Attendance vs marks correlation
- Impact of absenteeism on performance

### 🧠 Behavioral Analysis
- Login/activity frequency tracking
- Engagement vs performance correlation

### 📉 Data Visualization
- Bar charts
- Heatmaps
- Scatter plots
- Summary statistics tables

---

## 📂 Dataset Information

The dataset includes student academic and behavioral data such as:

- Student ID
- Student Name
- Marks / Scores
- Attendance Percentage
- Login / Activity Frequency
- Performance Category

---

## 🛠️ Tech Stack

- Python 
- Pandas 
- NumPy 
- Matplotlib 
- Seaborn 
- Jupyter Notebook 

---


## 📊 Key Insights

- Higher attendance generally leads to better academic performance
- Students with consistent engagement perform better in exams
- Low login/activity frequency correlates with poor scores
- Clear relationship between participation and academic success

## ⚙️ Installation & Usage

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Deb124-source/Student-Performance-Analysis-Dashboard.git
```

```bash
cd Student-Performance-Analysis-Dashboard
```

---

### 2️⃣ Create Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate environment:

#### Windows:

```bash
venv\Scripts\activate
```

#### Mac/Linux:

```bash
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Run the Project

```bash
jupyter notebook
```

Then open:

```text
notebooks/analysis.ipynb
```
